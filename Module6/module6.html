Object-oriented JavaScript

Object-oriented programming(oops) is a computer programming modal that organizes software design around data or object, rather than function and logic. An object can be defined as a data field that has unique attributes and behavior

The four pillers of oops are--

1-Encapsulation

2-Abstraction

3-Polymorphism

4-Inheritance

JavaScript is not actuallyy an object-oriented language, but we can write object-oriented code by using somthing called as a prototype object...

If the function that is being refrenced is a method in an object "this" refrences the object itself.
