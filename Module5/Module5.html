Functional programming?

Functional programming is programming paradigm designed to handle function in a better way in JS(mainly pure function).
JavaScript is multi-paradigm language. we can use object-oriented procedural and functional programming all the same time in JavaScript

Key feature of funtional programming

  • imperative and declarativ way of writing code

    • imperative--- how to go about a problem{approach, structure, logic}

    declarative--- what to achieve from the problem{only solution}

  • Pure function and what are side effects?
    • what is pure function-- apure function is a function which
    • Given the same input, always returns the same output.
    • produces no side effects

  • Higher order functions(map, filter, reduce, etc)
    • ---
    Higher order functions that operate on other functions, either by taking as arguments or by returning then
    in simple words, A higher order function is function that recieves a function as an argment or returns the as output.

  • composition
  • closure
  • currying
